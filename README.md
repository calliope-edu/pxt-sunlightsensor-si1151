# 🌞 SI1151 Sonnenlichtsensor-Erweiterung für MakeCode

Dies ist eine Beta-Version. Die UV-Index-Werte scheinen noch zu hoch zu sein.

Diese Erweiterung ermöglicht die einfache Ansteuerung des SI1151 Umgebungslicht- und UV-Sensors. Der Sensor misst sichtbares Licht, Infrarotstrahlung und UV-Index – ideal für Wetterstationen, Smart Farming, oder Umweltmessungen.

# 🔧 Funktionen

Auslesen von sichtbarem Licht (VIS) in Lux
Messen von IR-Licht (Infrarot)
Ermittlung des UV-Index
Einfache Integration in MakeCode-Projekte
Kompatibel mit I2C-Schnittstelle (Adresse: 0x53)

## 🌞 SI1151 Sunlight Sensor Extension for MakeCode

This is a beta version. The UV index values still appear to be too high.

This extension allows for easy integration of the SI1151 ambient light and UV sensor. The sensor measures visible light, infrared radiation, and UV index – ideal for weather stations, smart farming, or environmental monitoring.


# 🔧 Features

* Read visible light (VIS) in lux
* Measure IR light (infrared)
* Retrieve the UV index
* Easy integration into MakeCode projects
* Compatible with I2C interface (address: 0x53)

> Open this page at [https://calliope-edu.github.io/pxt-sunlightsensor-si1151/](https://jasperp92.github.io/pxt-sunlightsensor-si1151/)

## Use as an Extension

This repository can be added as a **MakeCode extension**.

* open [https://makecode.calliope.cc/](https://makecode.calliope.cc/)
* Click on **New Project**
* Click on **Extensions** in the gear menu
* Search for  **https://github.com/calliope-edu/pxt-sunlightsensor-si1151** and import it
